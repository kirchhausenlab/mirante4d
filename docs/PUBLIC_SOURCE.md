# Public Source Snapshot

Mirante4D public development begins from one deterministic, parentless Git
commit constructed from reviewed tracked files. Pre-public Git objects, refs,
workflow history, artifacts, caches, operator records, private data, and local
evidence are not part of this repository.

The root uses the generic identity
`Mirante4D Contributors <contributors@mirante4d.invalid>`, a fixed UTC
timestamp, regular mode `100644` unless explicitly reviewed executable, and the
message `Initial public Mirante4D source snapshot`. The lightweight
`foundation-public-root-v1` tag directly identifies that commit. The commit
cannot embed its own content-addressed ID; the tag and repository object graph
are the public readback authority.

Source lineage and human attribution are recorded in `README.md`,
`CITATION.cff`, and `ASSET_PROVENANCE.md`. The standard MIT license applies to
project-original source and fixtures; separately licensed retained material is
identified in the provenance notice.

The repository contains only small approved TIFF source fixtures. It contains
no full microscopy dataset and accepts no external dataset contribution.
