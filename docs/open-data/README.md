# Open Data

Mirante4D is being designed so that public microscopy datasets can eventually
be downloaded, integrity-checked, processed, and opened reproducibly.

No full microscopy dataset is currently released through this repository.
Local research data, private paths, subject/sample metadata, and unpublished
results must not be committed.

The foundation work may include small synthetic or rights-cleared fixtures for
format and reader tests. Full dataset publication is a separate follow-on that
must establish:

- scientific scope and selection;
- rights, privacy, and attribution;
- stable package and release identities;
- hosting and integrity checks;
- expected facts and reproducibility evidence;
- citation and versioning;
- explicit owner approval before upload or publication.

See [the deferred open-data plan](../plans/deferred/OPEN_DATA_FOLLOW_ON.md) for
the approved boundary.
