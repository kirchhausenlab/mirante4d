## Dependency-Provided Assets

The default UI font assets arrive through `epaint_default_fonts`. Cargo
metadata reports the package-specific `OFL-1.1` and `Ubuntu-font-1.0` licenses;
the narrow policy exception is documented in
[`docs/DEPENDENCY_EXCEPTIONS.md`](docs/DEPENDENCY_EXCEPTIONS.md). Release
packages also contain a generated `THIRD_PARTY_NOTICES.md` derived from the
exact locked Cargo graph.
